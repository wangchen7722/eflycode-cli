## 项目概述

Eflycode 是一个 AI 驱动的编程助手 CLI 工具，通过大语言模型帮助开发者提高编程效率。

### 主要特性

- **智能代码助手**：基于大语言模型的代码生成和辅助
- **工具集成**：内置文件操作、命令执行等工具
- **MCP 支持**：支持 Model Context Protocol 服务器集成
- **上下文管理**：智能的上下文管理和优化
- **交互式 CLI**：友好的命令行交互界面

### 系统要求

- Python >= 3.11
- 支持的操作系统：Linux、macOS、Windows
